# daystrack
An offline browser app for tracking time for events 

## Features
- **Event Management**: Create, edit, and delete events with custom titles and descriptions
- **Date Tracking**: Track days until or since events with date picker
- **Settings**: change light to dark theme, export or import data or even start a fresh

## Usage

### Adding Events
1. Tap the floating action button (+)
2. Enter event title and description
3. Select the date
4. Click "Save"

### Managing Events
- Tap on any event to view stats or even add/delete the event, previous logs etc.
